# Pickleball Tournament Planner

Organize a pickleball tournament in one page. Add players, generate a round
robin schedule, and track scores with automatic standings.

## Features

- Tournament setup fields for name, location, courts, and scoring rules.
- Player roster with shuffle and remove controls.
- Round robin schedule generation with court and slot labels.
- Live standings with wins, losses, ties, and point differential.
- Local browser storage to keep progress between visits.

## Getting started

Open `index.html` in your browser. For a local server, you can also run:

- `python3 -m http.server` in the `pickleball-tournament-app` folder.

## Data storage

All data is stored in the browser using local storage. Reset the tournament to
clear everything.
